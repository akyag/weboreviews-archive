Design

Underneath are a few modern concepts generally used in web design process. These established standards and practices will offer you an insight into what exactly is included in my web design services.

Web Standards

Web standards are the formal, non-proprietary standards and other technical specifications that define and describe aspects of the World Wide Web.

Web standards, in the narrow sense, consist of the following: Recommendations published by the World Wide Web Consortium (W3C), such as HTML, Cascading Style Sheets, the Document Object Model, image formats like Portable Network Graphics and Scalable Vector Graphics. My process includes these standardized best practices.

Responsive

Responsive web design is an approach to web design that makes web pages render properly on a variety of devices and window or screen sizes.

Your website will be crafted to provide an optimal viewing experience, easy reading and navigation with a minimum of re-sizing, panning, and scrolling across a broad range of devices (from mobile phones to desktop computer monitors).

Want to see it live in action? Try re-sizing your browser window!

Compatibility

Clean coding. Your website will function properly in all popular and latest browsers. I perform a cross-platform testing.

Accessibility

A website that is accessible is designed to function properly with assistive technologies so that the information and functionality that is fundamental to the website is made accessible to every visitor. I use a variety of tools to audit my websites and that also includes testing it against the web accessibility guidelines defined by W3C.

Contact Form

You will be provided a contact form with custom form fields, so your customers can contact you and provide information that you need.

Dark mode

The dark mode is a color scheme that uses light-colored text, icons and graphical user interface elements on a dark background. It reduces strain on your eyes and is very popular among all users. You can try seeing this website in dark mode if you haven't already. Use the theme button in the menu.

JS framework / CSS3 effects

Use of animation and rich functionality of JavaScript, providing ease of use to your visitors. I build websites using astro. You can find out more about the build tool from the link in the footer. Astro ensures your website is lightweight and only loads what's needed on the page.

No templates

I never use templates. Your website will be carefully crafted from scratch, just for you. That includes every small element. I don't use bootstrap or any other framework templates.

Optimization

When optimizing a website for performance, there are three primary types of front end optimization techniques.

  • Techniques that reduce the number of requests.
  • Techniques that reduce the overall size of the content.
  • Practices that promote parallelization (i.e. simultaneous download of assets).

By structuring the HTML properly, optimizing images, minifying CSS and JavaScript, your website will be lightening fast.
