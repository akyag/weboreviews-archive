What is a responsive design?

The goal of responsive web design is to make a web page look equally well regardless of the screen size of a device.

Design:

Web Standards

Your website will pass W3C validation. The Markup Validation Service is a validator by the World Wide Web Consortium (W3C) to check HTML documents for well-formed markup. Markup validation is an important step towards ensuring the technical quality of web pages.

Optimization

When optimizing a website for performance, there are three main types of front end optimization techniques.

  1. Techniques that reduce the number of requests.
  2. Techniques that reduce the overall size of the content.
  3. Techniques that promote parallelization (i.e. simultaneous download of assets).

By structuring the html properly, optimizing images through imagemin, minifying CSS through clean-css and Javascript through uglify-js, your website will be lightening fast.
